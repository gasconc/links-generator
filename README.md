## Installation

```
$pip3 install requirements.txt
```

## Quickstart

To use Link-Generator, follow these steps:

1. Data file: Copy&Paste the Data.xlsx file in the same path of main.py
2. Run the main file:

```
$python3 main.py
```

3. Check the output.xlsx file with your links
